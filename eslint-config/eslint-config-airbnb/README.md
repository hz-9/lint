## Installation

``` bash
npm install @hz-9/eslint-config-airbnb --save-dev
```

## Usage

Add to `.eslintrc.js`:

``` js
module.exports = {
  extends: [
    '@hz-9/eslint-config-airbnb'
  ],
}
```

Then run

``` bash
eslint --fix .
```
